# \#ProGuide



### \## Problem Statement

Students often struggle to receive proper career guidance from experienced professionals. Existing guidance system are either inaccessible, unstructured, or lack personalize mentorship.

ProGuide is a mentorship and career guidance platform where professionals can create profiles, define consultation fees, and provide mentorship sessions to students through appointment-based interactions.

The platform also includes an admin system to monitor users, appointments, and mentor activities.



### \##  User Roles:

\- Register/Login

\- Search mentors

\- View mentors profiles

\- Book mentorship sessions

\- View appointment history



### \## Professional / Mentor Roles:

\- Register / Login

\- Create mentor profile

\- Add expertise and experience

\- Set consultation pricing

\- Manage appointments



### \## Admin Roles:

\- Manage users

\- Monitor platform activities

\- Verify mentors

\- Manage categories



### \## Core Features

* JWT Authentication
* Role-Based Access Control
* Mentor Profile Management
* Appointment Booking System
* Mentor Search and Filtering
* Reviews and Ratings
* Admin Dashboard
* AWS S3 Profile Image Upload
* Responsive UI
